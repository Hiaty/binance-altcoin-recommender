# 🚀 币安山寨币智能推荐系统

基于币安Web3 API的智能加密货币分析工具，自动抓取实时数据，AI智能推荐。

## ✨ 功能特点

- 🤖 **全自动数据抓取** - 实时从币安API获取数据
- 📊 **K线形态分析** - 自动识别金叉、突破等信号
- 🎯 **庄家识别** - 检测持仓集中度、异常换手
- 💡 **智能推荐** - 多维度评分，推荐/观望/不推荐
- 🎨 **Web可视化界面** - 美观的交互式UI
- 📄 **分页展示** - 支持查看所有币种
- 🔍 **智能筛选** - 自动排除死币和低流动性币种

## 📖 使用指南

### 界面操作

1. **设置参数**
   - 筛选数量：选择分析多少个币种（5-50个）
   - 最小市值：设置最小市值（默认3000万）
   - 排序方式：按市值/评分/涨幅/换手率排序

2. **开始分析**
   - 点击「🔄 开始分析」按钮
   - 等待3-5分钟获取实时数据
   - 查看统计卡片和详细表格

3. **查看结果**
   - **推荐购买**：综合评分40+，建议关注
   - **建议观望**：评分20-40，需进一步观察
   - **不推荐**：评分<20，风险较高
   - **强庄币**：检测到庄家控盘迹象

4. **分页浏览**
   - 每页显示10个币种
   - 点击「上一页/下一页」查看所有币种

### 筛选条件

系统会自动过滤以下币种：
- ❌ 市值 < 3000万 或 > 10亿
- ❌ 24h交易量 < 50万（没交易量）
- ❌ 流动性 < 10万（流动性枯竭）
- ❌ 24h涨跌幅异常（-90%以下或+1000%以上）
- ❌ 已死亡的币种

### 分析指标说明

| 指标 | 说明 | 参考值 |
|------|------|--------|
| 历史最大回调 | 从最高点下跌幅度 | >50%为回调充分 |
| 底部最大反弹 | 从最低点上涨幅度 | 越高越好 |
| 换手率 | 24h交易量/市值 | 5-50%为健康区间 |
| 本周/本月反弹 | 7天/30天涨跌幅 | 反映短中期趋势 |
| 持仓集中度 | 前10大持有者占比 | <70%为筹码分散 |
| 日线/周线形态 | K线技术分析 | 识别买卖信号 |
| 资金净流入 | 买入量-卖出量 | 正值为资金流入 |
| 是否强庄币 | 庄家控盘迹象 | 是/否 |
