# 币安山寨币智能推荐系统 - 安装指南

## 🎯 目标
让小白用户也能轻松运行系统

## 📦 文件说明
- `一键启动.bat` - 双击运行（自动检测、安装、启动）
- `backend/` - 后端代码（无需修改）
- `frontend/` - 前端界面（无需修改）

## 🚀 使用步骤（只需2步）

### 第1步：安装Python（仅需一次）

**检查是否已安装：**
1. 双击运行 `一键启动.bat`
2. 如果显示"Python已安装"，跳到第2步
3. 如果显示"未检测到Python"，继续下面步骤

**安装Python：**
1. 访问 https://www.python.org/downloads/
2. 点击大大的黄色按钮 **"Download Python 3.11.x"**
3. 下载完成后**双击安装**
4. **⚠️ 重要**：安装界面底部勾选 **"Add Python to PATH"**
   （一定要勾选！）
5. 点击 **"Install Now"**
6. 等待安装完成

### 第2步：运行系统

1. 双击 `一键启动.bat`
2. 等待自动安装依赖（首次需要2-3分钟）
3. 浏览器自动打开，显示系统界面
4. 点击"开始分析"按钮使用

---

## ❓ 常见问题

### Q: 提示"pip不是内部命令"
**A**: Python安装时没勾选"Add Python to PATH"
- 重新安装Python，记得勾选！

### Q: 安装依赖时卡住
**A**: 网络问题，多等一会儿，或者换个网络试试

### Q: 浏览器没自动打开
**A**: 手动在浏览器输入 `http://localhost:5000`

### Q: 如何停止服务
**A**: 在黑色窗口中按 `Ctrl + C`，或者直接关闭窗口

---

## ⚠️ 重要提示

1. **只需要安装一次Python**，以后直接双击运行
2. **一定要勾选"Add Python to PATH"**，否则无法运行
3. **首次启动需要安装依赖**，会花费几分钟，请耐心等待
4. **确保网络畅通**，需要连接币安API获取数据
